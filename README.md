# TodoList

Full-stack приложение для управления задачами с CRUD-функциональностью.

## Стек

TypeScript, React, Zustand, Axios, Vite

## Установка

```bash
git clone https://github.com/Granit-700/todolist-client.git
cd todolist-client
npm i
```

## Запуск

Создать `.env` файл и вставить:
```
VITE_API_URL=http://localhost:3001
```
Команда для запуска:
```bash
npm run dev
```

> Для работы приложения нужен запущенный [бэкенд](https://github.com/Granit-700/todolist-server)
